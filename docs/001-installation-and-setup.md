# 001: Installation and Setup

To begin orchestrating horizontal "Journeys" within your enterprise Angular application, you must pull in both the core protocol and the Angular renderer.

## 📦 Requirements
- **Angular Environment:** 17+ (Strictly requires Native Signals support).
- **Control Flow:** Relies on modern built-in control flow (`@if`, `@for`).

## 📦 Installation

Install both the agnostic core schema package and the Angular-specific rendering wrapper.

```bash
npm install @slashand/sdui-blade-angular @slashand/sdui-blade-core
```
